## 📌 Feature
- [x] Vite + React + TypeScript
- [x] Tailwind CSS v4 (via `@tailwindcss/vite`, no PostCSS config needed)
- [x] Blank dark-theme page

## 📁 Project Structure
The directory structure of the project looks like this:

```
├── LICENSE
├── Makefile
├── README.md
├── index.html
├── package.json
├── vite.config.ts
├── tsconfig.json
├── tsconfig.app.json

└── src
    ├── main.tsx
    ├── App.tsx
    └── index.css
```

## 🚀 Getting Started

### Step 1: Install dependencies
```bash
npm install
# or
make install
```

### Step 2: Run the dev server
```bash
npm run dev
# or
make dev
```

Visit the printed local URL to see the blank dark page.

### Step 3: Build for production
```bash
npm run build
# or
make build
```
